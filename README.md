# Momentum

Momentum is an Android application designed to help users build momentum in life by tracking goals, habits, fitness journeys, career progress, and personal milestones.

## Features

- 🏃 Marathon Training Tracker
- 💼 Job Application Tracker
- ✅ Daily Tasks
- 📅 Calendar & Timeline
- 🔔 Smart Reminders
- 📈 Progress Analytics
- 🎯 Goal Tracking
- 📝 Journal
- 🤖 AI Coach (Planned)

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM Architecture
- Firebase
- Room Database
- Hilt
- WorkManager
- Material Design 3
